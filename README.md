# الذكي | Althaki - Smart Task Manager

The first application in the **Althaki (الذكي)** intelligent apps series.

## Overview

Althaki Smart Task Manager is a lightweight, elegant, and multilingual task management application focused on simplicity, intelligence, and productivity.

The application is designed with a modern Material 3 interface and is being developed incrementally with a strong focus on clean architecture, user experience, and future cross-platform support.

## Current Status

**Version:** Alpha 0.1

## Current Features

- 🌍 Automatic language detection (Arabic / English)
- ↔️ Automatic RTL / LTR support
- 📅 Live date and time from the user's device
- ✅ Smart task creation
- ⏰ Optional task start time
- 🔔 Optional reminder setting
- ✔️ Smart completion confirmation
- 🔄 Automatic rollover of unfinished tasks to the next day
- 💾 Local offline persistence
- 🎨 Material 3 modern interface
